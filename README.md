# Shaik Babasaheb — Portfolio

Personal developer portfolio built with **React**, **TypeScript**, and **Vite**.

🌐 **Live:** [shaikbabasahebdev.github.io](https://shaikbabasahebdev.github.io)

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React 18 |
| Language | TypeScript |
| Bundler | Vite |
| Styling | Custom CSS (CSS variables, dark/light theme) |
| Deployment | GitHub Pages |

---

## Features

- Animated typing hero with role carousel
- Scroll-reveal animations throughout
- Dark / Light theme toggle (persisted)
- Responsive layout — mobile, tablet, desktop
- Projects showcase with live site links
- Contact form that opens the default email client
- SEO meta tags + Open Graph / Twitter Card

---

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

---

## Project Structure

```
portfolio-ts/
├── public/             # Static assets (project screenshots, favicon)
├── src/
│   ├── components/     # React components (Hero, About, Skills, …)
│   ├── data/           # All content data (projects, skills, experience)
│   ├── hooks/          # Custom hooks (useTyping, useReveal, useCounter)
│   ├── styles/         # Global CSS
│   └── types/          # Shared TypeScript types
├── index.html
├── vite.config.ts
└── tsconfig.json
```
